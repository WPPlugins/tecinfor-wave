This is a plugin for feeding Adobe Wave. It sends a request to the feed when a blog post has been published.

== Description ==
**Português**
Este é um plugin para a alimentação Adobe Wave[wave.adobe.com](http://wave.adobe.com/). Ele envia um pedido à alimentação quando um post do blog foi publicado.

Importante<br>
Este plugin foi feito usando a API do wave.adobe.com - http://labs.adobe.com/technologies/wave/

**English**
This is a plugin for feeding Adobe Wave. It sends a request to the feed when a blog post has been published.

Important<br>
	
This plugin was made using the API wave.adobe.com - http://labs.adobe.com/technologies/wave/

== Installation ==

**Português**

1. Extraia o ficheiro na pasta `/wp-content/plugins/`.
1. Ative o plugin na parte de 'Plugins' do WordPress
1. Pronto!
	
	
**English**
	
1. Extract the file in the `/ wp-content/plugins /`.
1. Activate the plugin at the Plugins WordPress
1. That's it!
